# ai-health-fitness-agent
AI Health and Fitness Planning Agent – Streamlit Frontend
AI Health & Fitness Planning Agent – Milestone I
Overview

This project implements the front-end interface for an AI Health and Fitness Planning Agent. The goal of this milestone is to provide a simple user interface that allows users to input personal information and fitness goals and receive an initial recommendation generated by the system.

The interface collects user information such as age, sex, height, weight, activity level, and fitness goals. After submitting the form, the system calculates estimated calorie needs using a Basal Metabolic Rate (BMR) calculation and provides basic training and nutrition recommendations.

The purpose of this milestone is to demonstrate a functional front-end interface that allows interaction with the agent system and displays structured output in a way that is easy for a non-technical user to understand.

Technology Stack

The front-end interface was built using the following technologies:

Python – primary programming language
Streamlit – framework used to build the interactive web interface
GitHub – repository used for version control and code hosting
Streamlit Cloud – used to deploy the application as a web app

Streamlit was selected because it allows rapid development of interactive web interfaces with minimal front-end code while still producing a clean and accessible layout.

Interface Functionality

The interface currently allows users to:

Enter personal information such as age, sex, height, and weight
Select a primary fitness goal (fat loss, muscle gain, maintenance, etc.)
Choose activity level and training frequency
Provide equipment availability and diet preferences
Submit inputs to generate a response from the system

After submission, the system displays:

Estimated BMR (Basal Metabolic Rate)
Estimated daily maintenance calories (TDEE)
Suggested calorie intake based on the user’s goal
Protein intake recommendations
Basic workout structure suggestions

This output simulates the response from the agent system and demonstrates how users will interact with the application.

Current Scope of the Front-End

This version represents the initial front-end layer of the system. The focus of this milestone is to create a functional interface that gathers user inputs and displays calculated outputs.

At this stage, the application provides foundational calorie and training guidance but does not yet include the full functionality planned for the agent system.

Planned Improvements

Future versions of the system will expand the functionality of the agent by adding more detailed analysis and recommendations. Planned improvements include:

More detailed workout programming, including exercise selection and training splits
More comprehensive nutrition guidance including recommended intake of protein, fats, and carbohydrates
More personalized responses based on injury history or physical limitations
More detailed dietary recommendations and food suggestions
Integration with additional agents responsible for training planning, nutrition analysis, and safety checks

These improvements will allow the system to generate more advanced and personalized health and fitness plans.
